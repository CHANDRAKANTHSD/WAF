PROJECT OVERVIEW
----------------
A complete DDoS detection system using LightGBM and XGBoost machine learning
models. The system trains on multiple datasets, compares model performance,
and provides a REST API for integration into full-stack applications.

DATASETS USED
-------------
1. CICIDS2017 (3 files):
   - Friday-WorkingHours-Afternoon-DDos.pcap_ISCX.csv
   - Monday-WorkingHours.pcap_ISCX.csv
   - Thursday-WorkingHours-Morning-WebAttacks.pcap_ISCX.csv

2. CSE-CIC-IDS2018 (3 files):
   - DDOS attack-HOIC.csv
   - DDOS attack-LOIC-UDP.csv
   - DDoS attacks-LOIC-HTTP.csv

3. TON_IoT (1 file):
   - ton_iot_network.csv

KEY FEATURES
------------
✓ Sequential training on 7 datasets
✓ Automatic model comparison (LightGBM vs XGBoost)
✓ Feature selection (top 30 features)
✓ Class imbalance handling (SMOTE)
✓ Data normalization (StandardScaler)
✓ Performance metrics (Accuracy, Precision, Recall, F1-Score)
✓ Feature importance visualization
✓ Model export (.pkl and .joblib)
✓ REST API for deployment
✓ Inference time tracking
✓ Batch prediction support

QUICK START
-----------
1. Install dependencies:
   pip install -r requirements.txt

2. Verify setup:
   python test_setup.py

3. Train model:
   python ddos_detection.py

4. Test inference:
   python inference.py

5. Start API:
   python api.py

6. Test API:
   python test_api.py

WORKFLOW
--------
1. Data Loading
   └─> Load CSV files with proper encoding

2. Preprocessing
   ├─> Handle missing values
   ├─> Convert to numeric
   ├─> Remove infinite values
   └─> Binary label conversion (DDoS vs Normal)

3. Feature Engineering
   ├─> Feature selection (SelectKBest)
   └─> Feature normalization (StandardScaler)

4. Class Balancing
   └─> SMOTE oversampling

5. Model Training
   ├─> Train LightGBM
   ├─> Train XGBoost
   └─> Compare performance

6. Evaluation
   ├─> Accuracy
   ├─> Precision
   ├─> Recall
   ├─> F1-Score
   └─> Inference time

7. Model Selection
   └─> Choose best model based on F1-Score

8. Export
   ├─> Save model (.pkl, .joblib)
   ├─> Save scaler
   ├─> Save feature list
   └─> Generate feature importance plot

9. Deployment
   └─> REST API for real-time predictions

INTEGRATION OPTIONS
