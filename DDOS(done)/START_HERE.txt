================================================================================
                            TROUBLESHOOTING
================================================================================

Problem: "Model not found"
Solution: Run training first (run_training.bat)

Problem: "Dataset file not found"
Solution: Update paths in config.py

Problem: "Memory error"
Solution: Reduce N_FEATURES in config.py

Problem: "API connection refused"
Solution: Make sure API is running (run_api.bat)
